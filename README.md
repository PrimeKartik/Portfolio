## <a name="quick-start">🤸 Quick Start</a>

Follow these steps to run the portfolio locally:

**Prerequisites**

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/)

**Clone & Install**

```bash
git clone https://github.com/PrimeKartik/portfolio.git
cd portfolio
npm install
```

**Environment Variables**

Create a `.env` file in the root directory:

```env
REACT_APP_EMAILJS_USERID=your_emailjs_user_id
REACT_APP_EMAILJS_TEMPLATEID=your_emailjs_template_id
REACT_APP_EMAILJS_RECEIVERID=your_emailjs_receiver_id
```

Get your credentials from [EmailJS](https://www.emailjs.com/).

**Run**

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## <a name="project-structure">📁 Project Structure</a>

```
portfolio/
├── public/                  # Static assets (3D models, logo, etc.)
├── src/
│   ├── assets/              # Images and icons
│   ├── components/
│   │   ├── canvas/          # 3D components (Computers, Earth, Ball, Stars)
│   │   ├── About.jsx        # About / Overview section
│   │   ├── Contact.jsx      # Contact form with EmailJS
│   │   ├── Experience.jsx   # Achievements timeline
│   │   ├── Feedbacks.jsx    # Coding profiles section
│   │   ├── Footer.jsx       # Social links footer
│   │   ├── Hero.jsx         # 3D hero section
│   │   ├── Loader.jsx       # Loading spinner
│   │   ├── Navbar.jsx       # Navigation bar
│   │   ├── Tech.jsx         # Tech/skills 3D balls
│   │   └── Works.jsx        # Project cards
│   ├── constants/           # Data (projects, experiences, skills, etc.)
│   ├── hoc/                 # Higher-order components (SectionWrapper)
│   ├── utils/               # Motion animation utilities
│   ├── App.jsx              # Root app component
│   ├── index.css            # Global styles & gradients
│   ├── main.jsx             # Entry point
│   └── styles.js            # Tailwind style presets
├── index.html               # HTML entry point
├── tailwind.config.cjs      # Tailwind configuration
├── vite.config.js           # Vite configuration
└── package.json
```
